Unified-Web-Operations-Management-System


This project is a web-based Complaint Management System developed using the MERN stack. It provides a centralized platform where users can submit complaints and track their status, while administrators can manage and resolve them efficiently. The system is designed to improve communication, reduce manual work, and ensure transparency in complaint handling.

Features
User registration and login
Secure authentication using JWT
Submit complaints with category and description
Track complaint status (Pending, Resolved)
Admin dashboard to manage complaints
Update and delete complaints
Responsive and user-friendly interface
Technologies Used

Frontend: React.js, CSS
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT, bcrypt

Project Structure
client – frontend (React)
server – backend (Node.js, Express)
models – database schemas
routes – API routes
controllers – application logic
How It Works

Users register and log in to the system. After login, they can submit complaints with relevant details. The complaint is stored in the database with a status. The admin can view all complaints, update their status, and resolve them. Users can track the status of their complaints in real time.
